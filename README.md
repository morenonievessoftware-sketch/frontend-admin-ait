🎓 AIdeaTeach - Admin Panel

Panel administrativo de AIdeaTeach, plataforma educativa impulsada por IA para la generación de contenido pedagógico inclusivo basado en DUA y PIAR.

🚀 Descripción

Este frontend permite a los administradores gestionar:

👥 Usuarios (docentes)
📦 Planes de suscripción
💳 Pagos (integración con Wompi)
📊 Control general del sistema

Construido con una arquitectura moderna orientada a escalabilidad.

🛠️ Tecnologías
⚡ Vue 3 + Vite
🧠 Pinia (estado global)
🎨 TailwindCSS
🧩 PrimeVue
🔐 JWT Authentication
⚙️ Instalación
git clone https://github.com/tuusuario/aideateach-admin.git

cd aideateach-admin

npm install

▶️ Ejecución
npm run dev

🌐 Variables de entorno
VITE_API_URL=https://tu-backend.onrender.com

📦 Build
npm run build

🧠 Funcionalidades clave:
* Gestión de planes y precios
* Control de usuarios y roles
* Visualización de pagos
* Administración de suscripciones

🔐 Seguridad
Autenticación basada en JWT
Protección por roles (admin)
